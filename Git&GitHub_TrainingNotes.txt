1.  Git Configuration: One time thing
 
## Configure your user name
git config --global user.name "Stephanie Garcia"
 
## Configure your email address
git config --global user.email sjagarci@fiu.edu
 
## Configure MELD
git config --global diff.tool meld
git config --global difftool.prompt
 
## Check your already configured user name:
git config user.name
 
## Check your already configured user email:
git config user.email
 
## Check the detailed configuration of the loaded git on your machine
git config -l
